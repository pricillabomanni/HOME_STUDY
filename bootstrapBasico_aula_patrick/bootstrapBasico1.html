Paragrafos

o elemento HTML <p> (ou elemento de paragrafo) representa um paragrafo do texto.


Listas

O elemento HTML <ul> (ou elemento de lista desordenada) representa uma lista de itens sem ordem rigida, isto é, uma coleção de itens que não trazem uma ordenação numerica e as suas posições.


  • Banana
    • Nanica
    • Prata
  • Abacaxi
  • Graviola

O elemento HTML <ol> (ou elemento de lista ordenada) representa uma lista de itens ordenadas.

  1. Banana
  2. Aacaxi
  3. Graviola

O elemento HTML <dl> (ou elemento de lista de definições) engloba uma lista de pares de termos e descrições.

Texto pré-formatado

O elemento HTML <pre> representa o texto pré-formatado que deve ser apresentado exatamente como está no arquivo HTML. 

        ***********************
        * Texto pré-formatado *

Conteineres


Os conteineres são os elementos de Layout mais basicos no Bootstrap e são usados para conter, preencher e (às vezes) centralizar o conteúdo dentro.

Por padrão o bootstrap estabelece os pontos de quebra (media query) para alguns tamanhos de tela/janela.
